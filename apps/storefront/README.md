# 独立站

这是一个占位符目录。

## 部署选项

### Shopify Hydrogen
```bash
npm create shopify@latest storefront -- --template hydrogen
```

### Next.js Commerce
```bash
npx create-next-app@latest storefront --example commerce
```

### 其他方案
- WooCommerce (WordPress)
- Magento
- Saleor (开源)

---

部署后将 `apps/storefront` 替换为实际项目即可。
